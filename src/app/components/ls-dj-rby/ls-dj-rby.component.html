Django y Ruby en Linux Server

Guia de Configuracion de Django y Ruby en Linux Server

Para poder realizar esto primero tenemos que configurar usuarios y grupos.

Una vez teniendo los grupos y los usuarios definidos podemos realizar este tutorial.

Descargamos el sistema que queremos utilizar en nuestro servidor

Podemos ver que el usuario admin tiene todos los accesos

Utilizando los comandos "chmod" y "chown" para asignar usuarios y grupos para los accesos

Asi mismo con los mismo comandos se les asigna al otro sistema

Una vez asignados si tratamos de entrar con otro usuario que no es, nos denegara el acceso Al menos que estemos en el mismo grupo

Y con eso terminamos configuramos grupos y usuarios, y el sistema de nuestra eleccion en nuestro servidor de linux
