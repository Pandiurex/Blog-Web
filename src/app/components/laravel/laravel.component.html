Laravel

Laravel es un framework de código abierto para desarrollar aplicaciones y servicios web con PHP 5 y PHP 7. Su filosofía es desarrollar código PHP de forma elegante y simple, evitando el "código espagueti". Fue creado en 2011 y tiene una gran influencia de frameworks como Ruby on Rails, Sinatra y ASP.NET MVC. Laravel tiene como objetivo ser un framework que permita el uso de una sintaxis elegante y expresiva para crear código de forma sencilla y permitiendo multitud de funcionalidades. Intenta aprovechar lo mejor de otros frameworks y aprovechar las características de las últimas versiones de PHP. Gran parte de Laravel está formado por dependencias, especialmente de Symfony, esto implica que el desarrollo de Laravel dependa también del desarrollo de sus dependencias.

Caracteristicas Principales

Las características de Laravel son:
  • Sistema de ruteo, también RESTful.
  • Blade, Motor de plantillas.
  • Peticiones Fluent.
  • Eloquent ORM.
  • Basado en Composer.
  • Soporte para el caché.
  • Soporte para MVC.
  • Usa componentes de Symfony.
  • Adopta las especificaciones PSR-2​ y PSR-4.
  • Documentación incorporada accesible a través de la aplicación administrativa (incluyendo documentación generada automáticamente de los modelos y las bibliotecas de plantillas añadidas por las aplicaciones).
    • Laravel también es una plataforma habitual que brinda multiples herramientas

    Laravel Windows

    Para ver poder instalar Laravel, debemos abrir nuestro navegador y buscamos Laravel en nuestro servicio de busqueda preferida, ya sea Google, Yahoo, Bing, etc, yo les proporciono el link aqui: Laravel

    Tambien vamos a tener que descargar composer, aqui les dejo el link Composer

    Simplemente descargamos el instalador de composer, el cual sera la forma mas sencilla de trabajar con laravel

    Le damos siguiente al instalador y comenzara la instalacion de nuestro programa

    Aqui podemos ver como comienza la instalacion pero una vez que termine podremos acceder al siguiente paso

    Despues de que se instale tendremos esta consola

    Simplemente utilizamos este comando para instalar Laravel: composer global require laravel/installer Y listo ya tenemos Laravel en nuestro sistema

    Laravel en Linux

    Para instalar Laravel en linux usaremos la forma mas sencilla: y de la misma forma que en windows abrimos la pagina de Laravel

    Tambien tendremos que bajar composer de la misma forma que en windows

    Procederemos a instalar PHP, para eso tendremos que usar el siguiente comando sudo apt install php

    Ahora si proseguiremos a instalar composer con el siguiente comando: curl -sS "https://getcomposer.org/installer | php" Luego: sudo mv ~/composer.phar /usr/local/bin/composer Finalmente sudo apt install composer

    Ahora simplemente verificamos que composer este instalado poniendolo en la consola

    Ahora instalaremos laravel, para eso tenemos que usar los siguientes comandos: composer global require laravel/installer

    Ahora simplemente creamos un proyecto y levanatamos el servidor de la siguiente forma: composer create-project --prefer-dist laravel/laravel blog

    Ahora accedemos a la carpeta y levantamos el servidor

    Listo ya podemos trabajar con Laravel
